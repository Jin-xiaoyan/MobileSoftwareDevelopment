# Lab5 Art Space 实验报告
## 1. 应用展示内容
本应用为梵高油画艺术展示空间，主题是浏览经典美术作品，共展示**3幅梵高画作**，支持切换查看作品图片、名称、艺术家与创作年份。

## 2. 界面结构说明
整体使用**垂直Column布局**，分为3个区块：
1.  作品图片区：Surface+Image组件，带画框效果
2.  作品说明区：嵌套Column+Text组件，展示文字信息
3.  切换控制区：水平Row布局，放置Previous、Next两个Button按钮
通过Modifier、Spacer控制间距与对齐，完成组件嵌套排版。

## 3. Compose状态管理
使用`remember+mutableIntStateOf`定义`currentArtwork`索引状态，默认值为1。状态改变时，UI会自动刷新，动态加载对应图片与作品信息。

## 4. 按钮条件逻辑（when表达式实现）
- Next按钮：索引+1，第3幅点击循环回到第1幅
- Previous按钮：索引-1，第1幅点击循环回到第3幅

## 5. 问题与解决
1.  图片与代码索引不匹配：重新对应文件名修改映射逻辑
2.  界面排版混乱：按实验拆分3个区块，规范Row/Column布局
3.  首尾无法循环：使用when表达式完善边界循环逻辑
4.  状态不刷新：正确使用remember状态委托语法
